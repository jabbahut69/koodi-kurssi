HTML perusteet

Luentomuistiinpanot

Ensimmäiset luennot menivät käytännön asioiden selvittelyyn, keskusteluun ja luentoihin internetin ja nettisivujen perusteista. Minulla ei ollut aikaisemmin näistä hirveän hyvää käsitystä, mutta nyt ensimmäisen viikon jälkeen on jo niistä parempi fiilis. Sain ruksattua skill-puusta kaikenlaista yli ja ruksailusta tuli onnistunut olo, kun huomaa mitä kaikkea sitä tietää ja osaa. Lisäksi omana tavoitteena kurssille on mm. oppia tekemään samanlainen konfetti-efekti, joka tulee kun saa ruksattua jonkin skillin yli. Siitä tulee niin hyvä fiilis ja tekee mieli naksutella painikkeita edes takas.

Alla lista muistiinpanoista, jotka tein koodiin, mutta siirsin tähän koska ne oppi hahmottamaan aika nopeasti ja tulivat vaan tielle kun rupesi muokkaamaan koodia enemmän. Hyvä kuitenkin ehkä jossakin pitää ainakin aluksi kirjallisesti.

  • CSS = CASCADING STYLE SHEETS. Koodi lukee vasemmalta oikealle, ylhäältä alas, eli alempana oleva koodi jää voimaan, jos määritellään samaa asiaa useampaan kertaan. Classit override yleisellä tasolla määritellyt ominaisuudet. Id on kaikista korkeimmalla hierarkiassa.
  • HTML kommentti on < !----> (väliin ei kuulu välilyöntiä, mutta muuten koodi piilottaa) ja CSS /* */
  • Class on atribuutti, jolla voidaan identifioida tiettykä elementtejä HTML:ssä, class merkitään pisteellä kun määritellään sen tyyliä CSS:llä.
  • Div on yleiselementti, ja sen sisään voi laittaa kaikenlaisia juttuja (esim. tekstiä ja kuvia). On kuin laatikko.
  • Lang-elementti kertoo selaimelle sivun oletuskielen (mm. tavutuksen apuna, osaa oikeat merkit jne.)
  • Br (break) attribuuttia ei suljeta
  • DOCTYPE html-elementti on HTML dokumentin juurielementti, HTML-tiedosto tarvitsee sellaisen tiedoston ihan alkuun.
  • Img-tagi on kuville (muista lisätä src attribuutti), kuvatiedoston voi laittaa vaikka keskelle tekstiä. Asettelua voi muuttaa muuttamalla block elementiksi.
